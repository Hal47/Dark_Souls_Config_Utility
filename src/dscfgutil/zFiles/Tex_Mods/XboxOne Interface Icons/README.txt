Xbox One Interface Icons v1.0

Installation:
=============

— Unzip directly to Dark Souls: Prepare To Die Edition's install folder or simply copy 40fbc4ad40fbc4ad.png to:

*\Steam\steamapps\common\Dark Souls Prepare to Die Edition\DATA\dsfix\tex_override

— Make sure "enableTextureOverride" in DSfix.ini is set to "1", else it won't work.
